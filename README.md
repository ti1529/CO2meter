# README

## 開発言語
- Ruby 3.3.0
- Rails 7.2.2.1
- PostgreSQl 14.15
  
## 実行手順
```
$ git clone https://github.com/ti1529/co2meter_app.git
$ cd co2meter_app
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
```
